# Quiz Game (MVC)

Simple, basic ASP.NET Core MVC quiz game with login/registration, timed quiz (Easy/Hard), shuffled questions, score saving, and a leaderboard along with Feedback.

## Requirements
- Visual Studio 2022+
- .NET SDK (same version as the project target framework)
- MySQL server

## MYSQL:
- "quizgame"- DATABASE
- "users","scores","feedback"- tables
